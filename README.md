# Journal

A personal journal built with React Router.

**Demo:** https://journal.razik.net

## Development

```bash
bun install
bun run dev
```

## Production

```bash
bun run build
bun run start
```

## Docker

```bash
docker build -t journal .
docker run -p 3000:3000 journal
```

Or pull from GHCR:

```bash
docker pull ghcr.io/razikdontcare/journal:latest
```
